# DNTLVCRMBS Pizza Club

Official website for Don't Leave Crumbs Pizza Club — Atlanta's Friday-night artisanal pizza ritual in Grant Park.

## Local development

Just open `index.html` in any browser. No build step required.

## Deployment

This is a static site. Deploy to Vercel, Netlify, Cloudflare Pages, or any static host:

```
vercel deploy --prod
```

## Files

- `index.html` — full site
- `fonts/FiftiesMovies.ttf` — custom logo/display font
